# troubleshooting-wifi-lemot
Project ini berisi pengalaman saya dalam menangani masalah WiFi lemot pada jaringan.
WiFi sering mengalami penurunan kecepatan saat digunakan banyak user, terutama saat jam sibuk.
Permasalahan:Koneksi lambat Ping tinggi Kadang putus nyambung
Analisa Beberapa kemungkinan penyebab:
Terlalu banyak user, Bandwidth tidak dibatasi ,Interferensi sinyal, Router terlalu panas
Solusi yang Dilakukan:
Membatasi bandwidth per user,Restart router secara berkala,Mengatur posisi router,Mengecek penggunaan jaringan
Hasil Setelah dilakukan perbaikan:
Koneksi lebih stabil,Ping menurun,User lebih nyaman
Tools:
Mikrotik (Winbox),Speedtest,Ping test
